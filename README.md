# HanMemo Backend API

Spaced Repetition System (SRS) for Mandarin Chinese vocabulary learning.

## Tech Stack
- **Node.js** + Express.js
- **MySQL** + Sequelize ORM
- **JWT** authentication
- **SM-2** algorithm for SRS

## Quick Start

### Prerequisites
- Node.js 16+
- MySQL 8+
- npm

### Installation

```bash
cd server
npm install
```
